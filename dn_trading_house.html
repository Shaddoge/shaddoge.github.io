Dragon Nest Trading House System

Project Description

This group project is to replicate the Trading House System of the game Dragon Nest while using a custom backend server that the group has made. The user also has the ability to edit an existing player's name if they are not an admin and they are able to create a new player, item, and inventory item for a player if they are an admin.

This group project was made using MySQL, TreeFrog, Strawberry GraphQL, C++ and Unreal Engine 4 with VaRest plugin for our course: Information Management in Games.
