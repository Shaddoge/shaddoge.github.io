Super Mario Bros. Recreation

Project Description

Super Mario Bros. Recreation is based from a famous retro platform game "Super Mario Bros.". This project has first three levels and displays the top score in the main menu. This project practices object-oriented programming with different entities in the game and utilizes SFML's colliders to create interactions between mario's environment and enemies.

This group project was made using C++ and SFML for our course: Advanced Concepts for Game Programming.

Role In Project

My role in this project is Programmer and Level Designer as I create techniques in order to create the levels by loading a file since the game is grid based.
