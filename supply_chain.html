Anno 1800 Supply Chain Management

Project Description

This project is to replicate one of the Anno 1800's suppy chain management system, and that is to produce sewing machine from the factories. This is to try and produce sewing machine with efficient time by ordering the vehicles which to pick items to pick up or not. There are factories that can produce base resources with limited outputs: iron mine, coal mine, and lumberjack's hut. The materials being outputted will be loaded by trucks to unload to their destination (lumbermill (furnace or sewing machine factory) to in order to make a sewing machine.

This project was made using C++ and Unreal Engine 4 for the course: Game Engine 1.
