Apartment Furnishing

Project Description

A studio apartment made from an empty room template provided from the course. Our task is to populate the empty room and make our own version of the apartment.

This group project was made using and Unreal Engine 4 for our course: Game Engines 1

My role for this project is Level Designer as I continue to apply proper textures, add models for the scene. The tasks are equally splitted between the members.
