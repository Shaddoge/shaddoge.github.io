Elemental Breakout

Project Description

Elemental breakout is a 3D mobile rail-shooter with three playable levels that uses specialized inputs like accelerometer. Defeat enemies according to their elemental weakness to deal much more significant damage and purchase permanent player upgrades for faster level clears.

This group project was made using C# and Unity for our course: Web and Mobile Game Development.

Members

  • Omalin, James Gabriel
  • Santos, John Adriel

Video Demo

Role In Project

My role in this project is Programmer, Level Designer, and Game Designer. Since there are only two members for this project, we had to divide tasks and I had to focus more on the frontend and handle most functionality beside the core mechanics. I had to work on the ads, app notifications, web request for leaderboard, UI, and designing all levels I oversee the progress of the game build through utilization of source control.

Specific Distributions

  • Forest, Desert, and Tundra design
  • Specialized inputs: swipe, pinch, spread, accelerometer
  • Enemy waves balancing
  • Score system
  • Notification system
  • Ad system: banner, interstitial, reward
  • Leaderboard with web handlers
  • User Profile: add, edit, delete
  • Settings
  • UI: game over screen update, main menu, leaderboard, user profile, settings, how to play
