Azure Chronicles

Project Description

Azure Chronicles is a 3D action RPG game serious game that aims to raise awareness on the degradation of coral reefs in the Philippines. The player will play as the Pearlescent Reefs' Ancestral Protector which has been constantly summoned by their community and must restore balance in the great reefs through investigations and taking care of it. It aims for learning to come naturally while exploring the world around them through storytelling and the visual environment.

This group project is for our Capstone where we are tasked to create a serious game that tackles one of the (SDGs) Sustainable Development Goals. It is currently in development and expected to release before the end of August 2023.

Role In Project

My role for this project is Lead Developer and worked around both backend and frontend and provided ease of use for the reusable functions that can be used by other developers. I have developed the core game mechanics, and other features that makes up the serious game an education and fun at the same time. Specific Distributions

  • Systems: Dialogue, Quest, Runic Stone, Charm, Codex, Save & Load, Enemy Spawn

  • Character State Machine for movement, combat, and utilities
  • Time Bubble Mechanic
