Herb Buddies

Project Description

Herb Buddies is a tower defense game where the player clears levels challenged by waves of enemy hoards to finish the game. The player will need to explore the map, gather crafting materials and build turrets to fend off all the monsters and, ultimately, find a way out.

This group project was made using C# and Unity for our courses: Project 1, Project 2, Project 3.

Role In Project

My role for this project is Programmer and Lead Designer and worked around both backend and frontend. I have developed some of the core game mechanics, and other features for the game loop. I also oversee the progress of the game build through utilization of source control and make sure that every feature implemented by other developers are working properly

Specific Distributions

  • Inventory system
  • Dialogue system
  • Item spawn and randomization
  • Enemy wave and pathing system
  • Essence Generator system
  • Enemy types & behaviors
  • Room unlocking
  • Camera system
  • Scene Lighting
  • Game pausing
  • Room building for all levels
  • Sound system
  • UI Functionality: HP bar, inventory, pause, main menu, key, level clear
  • VFX: tutorial glow, essence generator, door barrier unlocking, end door unlocking, turret building smoke, turret firing, enemy portal, enemy wave anticipation
