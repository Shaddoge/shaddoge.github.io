Virtual Museum

Project Description

This virtual museum project is a showcase where the scene is surrounded by skybox, and in the middle consists of models with applications of texture/multitexture with normal maps affected by different types of light such as point, spot, and directional. The scene also has a day/night cycle which affects the lighting of the scene in realtime. The user is in a 1st person view and may navigate the virtual museum by using WASD and mouse controls.

This group project was made using C++ and OpenGL for our course: Introduction to Computer Graphics.
