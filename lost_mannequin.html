Lost Mannequin

Project Description

Waking up as a mannequin without your limbs in a strange place, you wonder how you will escape. Come and join the mannequin in an adventure of looking for their limbs in this mysterious ruin and escape.

This group project was made using C# and Unity for our elective course: Game Development Fundamentals.

Role In Project

My role for this project is Lead Developer and worked around both backend and frontend and provided ease of use for the reusable functions that can be used by other developers. I have developed most of the core game mechanics for the completion of the game. I oversee the progress of the game build through utilization of source control and make sure that every feature implemented by other developers are working properly

Specific Distributions

  • Utilized event system for game mechanics, dialogue system, and user interface
  • Implemented item push/pull/grab through the use of ray casting.
  • Puzzle mechanics: button interactions, button light switch for torch room, pressure plate activation, opening doors, moving platforms
  • Easter eggs
  • Ending
  • UI: main menu, pause, end credits
